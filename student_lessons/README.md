# Student Lessons

`student_lessons/` is the student-facing lesson shelf for this repo.

It stores explanations of concepts that have already been taught or are being introduced on purpose before the next task.

## How To Use It

- start with `index.md`
- read only the lesson that matches the current task
- use lessons as references, not as answer keys for assessment prompts

## What Belongs Here

- short fundamentals primers
- concept explanations that were already taught in chat
- reusable notes for future tasks

## What Does Not Belong Here

- dated session logs
- private teacher assessment notes
- full solutions copied from baseline exercises
