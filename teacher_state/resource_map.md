# Resource Map

- Python Tutorial: Control Flow Tools  
  https://docs.python.org/3/tutorial/controlflow.html  
  Use when the student needs a focused refresher on `if`, loops, function parameters, or `return`.

- Python Tutorial: Data Structures  
  https://docs.python.org/3/tutorial/datastructures.html  
  Use when list or dictionary operations are the blocker in a concrete task.

- Built-in Types Reference  
  https://docs.python.org/3/library/stdtypes.html  
  Use for exact method behavior such as `str.split`, `str.strip`, `str.join`, or mapping behavior like `dict.get`.

- Python Tutorial: Errors And Exceptions  
  https://docs.python.org/3/tutorial/errors.html  
  Use when teaching how to read failures, understand `ValueError`, or interpret tracebacks.

- `unittest` Library Reference  
  https://docs.python.org/3/library/unittest.html  
  Use when the student needs to understand what a test is asserting and how tests drive iteration.
