# Lint Checklist

- Do the current `teacher_state/` pages agree with each other?
- Did any recent session note introduce a conclusion that is missing from `teacher_state/`?
- Is `next_task.md` still the smallest useful next step?
- Are any hypotheses stale, contradicted, or missing confirm/disconfirm criteria?
- Does `resource_map.md` contain only motivated, high-value links?
- Are any canonical pages empty, redundant, or drifting into journal mode?
- Is the baseline still solution-free and evidence-generating?
- Append a `lint` entry to `log.md` when a pass changes the repo or resolves ambiguity.
