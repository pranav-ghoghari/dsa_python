# Canonical Teacher Wiki

`teacher_state/` is the current best synthesized model of the student.

These files are not a dated journal.

## Rules

- Update these files in place when better evidence appears.
- Prefer revising stale beliefs over accumulating contradictions.
- Keep entries concise, evidence-based, and decision-useful.
- Separate observed facts from active hypotheses.
- Keep `next_task.md` to exactly one concrete next task.
- Update `index.md` and `log.md` whenever meaningful changes are made.
- When a canonical page is added, renamed, merged, or removed, update any affected cross-references in addition to `index.md` and `log.md`.
- When asked "what next?", read this directory before historical notes.
- File durable analyses back into this directory when they have ongoing value.

## Maintenance

- Use `lint_checklist.md` for periodic consistency checks.
- If a page is empty or redundant, improve or merge it rather than letting dead files accumulate.
