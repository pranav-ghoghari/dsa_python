# Historical Evidence Notes

This directory stores dated session evidence.

It is historical and append-oriented.

## Rules

- Put dated notes in `sessions/`.
- Keep one note per meaningful session, review, or update event.
- Keep notes brief, concrete, and evidence-based.
- Record what happened, what changed, and what teaching implication followed.
- Do not fabricate dates or evidence.
- Do not let this directory become the only memory system.

## Relationship To `teacher_state/`

- Major conclusions belong in `teacher_state/`, not only here.
- When a session note changes the current student picture, update `teacher_state/` in the same turn.
- Preserve old notes as history even when the current canonical view changes.
