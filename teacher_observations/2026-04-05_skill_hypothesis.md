# Current Skill Hypothesis

This is a working hypothesis, not a final judgment.

## Likely Current Level

- Can often understand code intent in plain English.
- Can decompose a simple task into steps.
- Can recognize common programming concepts such as:
  - function
  - argument
  - list
  - dictionary
  - boolean
  - loop
  - return value
- Cannot yet reliably produce correct Python syntax from memory.
- Can sometimes guess the right operation family, but not always the right method or exact expression.

## Likely Strengths

- Conceptual reasoning
- Reading intent from code and comments
- Mapping business behavior to code behavior

## Likely Weaknesses

- Syntax recall
- Knowing which built-in string/list/dict operations exist
- Turning an idea into exact Python statements
- Confidence when facing a blank file
- Distinguishing "I know the concept" from "I know the correct Python tool for it"
- Reading test infrastructure and error-reporting concepts

## Teaching Implication

- Do not begin with algorithms or abstract computer science vocabulary.
- Begin with short, concrete Python drills:
  - variables
  - strings
  - lists
  - dictionaries
  - loops
  - `if`
  - functions
  - `return`
- Use tiny exercises with immediate feedback.
- Require the student to type code, not just describe it.
- Use the student's comments in code as a bridge from natural-language reasoning to valid syntax.
